# Masar brand & Arabic conventions

"مسار" is an Arabic personal time + religious tracker. Sibling artifacts (e.g. the promo video) must match the same identity.

- **Palette:** background black `#0A0A0B`, gold `#C9A24B` (primary accent), teal/turquoise `#5FA8A0` (secondary).
- **Language/direction:** all user-facing copy is Arabic, RTL.
- **Text rule:** do NOT use the em-dash (—) in Arabic UI text.

**Why:** The user explicitly set these constraints and reuses them across artifacts; matching brand is a stated requirement, not a creative choice.

**How to apply:** When building any new مسار-related surface (web, video, slides), pass these as fixed constraints. Core product pillars to convey: prayer tracking, azkar/istighfar, Quran progress, daily tasks/commitments, focus sessions, points/levels gamification, reports, optional cloud sync — the value prop is balancing dunya (productivity) and akhira (worship) in one app.
