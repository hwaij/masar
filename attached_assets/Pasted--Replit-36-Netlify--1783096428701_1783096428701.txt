الموقع على Replit يستهلك فلوس كثير ($36 هذا الشهر).

الحل: انقل الموقع كاملاً لـ Netlify (مجاني):

═══════════════════════════════════════

1. بناء الكود (Build):
   - في Replit: شغّل `npm run build`
   - تأكد من البناء نجح (بدون أخطاء)
   - الملف النهائي يكون في `/dist` أو `/build` حسب الإعدادات
   - هذا الملف اللي بينرفع لـ Netlify

2. رفع لـ Netlify:
   - اذهب netlify.com
   - اضغط "New site from Git"
   - اختر GitHub و authorize
   - اختر repository: hwaij/masar
   - في Build settings:
     * Build command: `npm run build`
     * Publish directory: `/dist` (أو `/build`)
   - Deploy!
   - Netlify بيعمل auto-deploy عند كل push لـ GitHub

═══════════════════════════════════════

3. احذف كل استدعاءات Anthropic API (مكلفة جداً):
   
   a) في lib/helpers.js:
      - احذف دالة `analyze()`
      - احذف دالة `coachChat()`
   
   b) في جميع Components:
      - ابحث عن استدعاءات analyze أو coachChat
      - احذفها
      - أمثلة:
        * في AzkarPage.tsx - ابحث عن "اقترح تحديات" - احذفه
        * في AnalysisPage.tsx - ابحث عن "ابدأ التحليل" - احذفه
        * في أي مكان فيه "مساعد" - احذفه كاملاً
   
   c) احذف أي زر أو أيقونة فيها ✨ (نجمة بريق)
      - البحث عن: `✨` أو `sparkles` أو `Sparkles`
      - احذفها من الواجهة

═══════════════════════════════════════

4. بدل Anthropic بـ Google Gemini (مجاني):
   
   a) احصل على Gemini API Key:
      - اذهب console.cloud.google.com
      - Create new project
      - Enable "Generative Language API"
      - Create API Key
      - نسخ الـ key
   
   b) في الكود:
      - أنشئ ملف جديد: `lib/gemini.js`
      - اكتب دالة استدعاء Gemini:
        ```javascript
        export async function getGeminiResponse(prompt) {
          const response = await fetch(
            'https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=' + GEMINI_KEY,
            {
              method: 'POST',
              headers: { 'Content-Type': 'application/json' },
              body: JSON.stringify({
                contents: [{ parts: [{ text: prompt }] }]
              })
            }
          );
          return response.json();
        }
        ```
   
   c) أضف Gemini في مكان واحد بسيط:
      - مثلاً في Dashboard أو Today page:
        * زر: "اقتراح تحدي اليوم" (استدعاء Gemini)
        * أو "ملخص سريع ليومك"
      - استدعاء Gemini مرة واحدة فقط
      - بدون ضغط على المستخدم

═══════════════════════════════════════

5. البيئة والإعدادات:
   
   a) متغيرات البيئة (Environment Variables):
      - في Netlify: Site settings → Build & deploy → Environment
      - أضيف: VITE_GEMINI_API_KEY = [API Key]
      - في الكود: استخدم `import.meta.env.VITE_GEMINI_API_KEY`
   
   b) في GitHub:
      - ما تضع API Key مباشرة
      - استخدم environment variables
